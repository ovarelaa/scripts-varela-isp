# scripts-varela-isp
Scripts criados para facilitar o atendimento do CSA no ISP Integrator.

abertura-atendimento.js:
Script feito para agilizar a abertura e troca de motivo de atendimento, com botões adicionais no ISP com os motivos mais utilizados e tipos de atendimentos.

auto-sz.js:
Para evitar manter atendimentos pendentes no cargo, este script altera o status para "Aguardando informações", seleciona a categoria "Técnico" e por fim, adiciona um comentário informando que o cliente está com você via SZ Chat.

copiar-atendimentos-isp.user.js
Para evitar descontos na monitoria, este script copia todos os atendimentos do cliente nos últimos 3 meses e especifica o status de cada um dos atendimentos. Além disso, também retira o pop-up clicável de ter copiado os atendimentos, aparecendo apenas momentaneamente.

documentação-ax2-ws.js
Script que puxa todos os dados de roteadores HUAWEI AX2, V2 e V3 automaticamente para seu bloco de notas, agilizando na tratativa.
